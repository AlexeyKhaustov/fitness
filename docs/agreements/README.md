# Документация проекта FitnessVideo

Добро пожаловать в документацию проекта. Здесь собраны материалы для разных аудиторий:

- **[Правовая документация](legal/compliance.md)** — описание процедур сбора согласий, хранения персональных данных, версионирования документов. Предназначена для проверяющих органов и юристов.
- **[Техническая документация](technical/consent_system.md)** — архитектура системы согласий, модели, middleware, views, URL. Для разработчиков, которые будут поддерживать или дорабатывать проект.
- **[Информация для пользователей](user/consent_info.md)** — краткое объяснение, зачем нужны документы, как даётся согласие и что происходит при их обновлении.
- **[Инструкция для администраторов](admin/document_management.md)** — как управлять версиями документов через админку.

По всем вопросам обращайтесь к команде разработки.
